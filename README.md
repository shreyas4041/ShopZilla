# ShopZilla - E-Commerce Platform

ShopZilla is a modern e-commerce platform with a bold black, white, and orange design. It offers a seamless shopping experience across all categories, from electronics to fashion.

## Running the Application

This is a frontend-only application that runs directly in the browser. It uses a mock API that leverages browser `localStorage` to persist data.

To run the application, you can use a simple static file server or open the `index.html` file directly in a modern web browser.
